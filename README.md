## Installation

**Placeholder**: August 2024 - the package is in alpha (re)development and not yet up to date on `pypi`

With `pip`:

```bash
pip install efts-io
```

With [`pipx`](https://github.com/pipxproject/pipx):

```bash
python -m pip install --user pipx
pipx install efts-io
```
